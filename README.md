# is-audio
Check if a filepath is a audio file

## Install

```
$ npm i @kikopalomares/is-audio
```


## Usage

```js
const isAudio = require('@kikopalomares/is-audio');

isAudio('src/yourFile.mp3');
//=> true

isAudio('src/yourFile.txt');
//=> false
```
